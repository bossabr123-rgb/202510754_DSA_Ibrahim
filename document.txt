origin.txt
Plain text
Chapter 1: Origin of C++

C++ was developed by Bjarne Stroustrup in 1979 at Bell Labs.

The language started as an extension of the C language and
was originally called "C with Classes".

Its goal was to add object-oriented programming features
while maintaining the efficiency and performance of C.
development.txt
Plain text
Chapter 2: Development of C++

C++ introduced many important programming concepts:

- Classes
- Objects
- Encapsulation
- Inheritance
- Polymorphism

These features made software development easier,
more organized, and reusable.

Over time, C++ became one of the most popular
programming languages in the world.

Arrays in C++

An array stores multiple values

filestreams.txt
Plain text
File Streams in C++

File handling allows programs to read
and write information from files.

Types of file streams:

ifstream  - Read data from a file
ofstream  - Write data to a file
fstream   - Read and write data

Example:

ifstream file("data.txt");

string line;

while(getline(file, line))
{
    cout << line << endl;
}
